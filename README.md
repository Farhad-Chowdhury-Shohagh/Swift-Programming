## 📌 About This Repository

This repository contains simple and practical Swift programs designed for beginners. Each project focuses on basic programming concepts such as variables, data types, conditions, loops, arrays, functions, classes, and user input handling.

The main goal of this repository is to learn Swift step by step through small console-based projects.

## 🚀 Features

- Beginner-friendly Swift programs
- Clean and easy-to-understand code
- Console-based mini projects
- Practice examples for Swift fundamentals
- Useful for students and new Swift learners
- Organized project structure

## 🛠️ Technologies Used

- **Programming Language:** Swift
- **Code Editor:** Visual Studio Code / Xcode
- **Platform:** macOS, Windows, or Linux with Swift installed
